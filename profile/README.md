# Welcome to the SoCLabs git repository

## Background

SoC Labs is the global academic community for System On Chip development using the Arm based ecosystem. The core activity of SoC Labs is community driven SoC design projects. Full details can be found on the [SoCLabs community portal](https://soclabs.org/)

## Repository Structure
The main goal of SoC Labs is to create reusable reference designs, the repositories for these have been seperated into 3 different categories:

- Projects: top level projects that encompasses Tech and Flow subrepositories
- Tech: subrepositories for technology rtl
- Flows: subrepositories for flows (simulations, fpga, backend)

For example, the [nanosoc](https://soclabs.org/reference-design/nanosoc) reusable reference design has the following structure

- Top level: [nanoSoC Project](https://github.com/SoC-Labs/nanoSoC-Project)
    - Nanosoc Tech: main RTL for nanosoc
    - SoCTools Flow: Toolkit for SoC Labs projects

You will also find top level projects for megaSoC and milliSoC
